# Real-Time AI Object Detection and Tracking System

## 📌 Project Overview

This project uses YOLOv8 and OpenCV to perform real-time object detection and tracking using a webcam. The system can detect and track objects such as persons, mobile phones, bottles, laptops, and more in real time.

---

## 🚀 Features

* Real-time object detection
* Object tracking using YOLOv8
* Person counting
* FPS (Frames Per Second) display
* Webcam-based live detection

---

## 🛠 Technologies Used

* Python
* OpenCV
* YOLOv8
* Ultralytics
